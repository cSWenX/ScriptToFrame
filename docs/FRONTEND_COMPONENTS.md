# 前端组件文档

## 组件结构

### `/components/ScriptInput.jsx`
剧本输入组件
- 大文本框
- 格式验证
- 实时字数统计

### `/components/ControlPanel.jsx`
控制面板组件
- 关键帧数量滑块 (3-12)
- 画风选择
- 生成按钮
- 进度显示

### `/components/StoryboardDisplay.jsx`
分镜图显示组件
- 卡片式布局
- 图片预览
- 标注信息
- 提示词显示
- 重新生成按钮

### `/components/FirstImagePreview.jsx`
首张图预览组件
- 大图预览
- 确认/重新生成按钮
- 风格调整选项

## 布局说明

### 三栏布局 (30% - 15% - 55%)
- **左栏**: 剧本输入区
- **中栏**: 控制选项区
- **右栏**: 结果展示区

### 响应式设计
- 桌面端三栏布局
- 移动端单栏堆叠

## 交互流程

1. 用户输入剧本
2. 选择生成参数
3. 点击"生成第一张图"
4. 确认满意后点击"生成剩余图片"
5. 查看完整分镜结果
6. 可单独重新生成任意图片
