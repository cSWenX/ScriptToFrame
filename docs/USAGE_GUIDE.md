# 安装和使用说明

## 快速开始

### 1. 环境准备
确保已安装 Node.js 18+ 和 npm

### 2. 安装依赖
```bash
cd script-to-frame
npm install
```

### 3. 配置环境变量
复制环境变量模板并填入真实API密钥：
```bash
cp .env.example .env.local
```

编辑 `.env.local` 文件：
```env
VOLCENGINE_ACCESS_KEY_ID=你的火山引擎访问密钥ID
VOLCENGINE_SECRET_ACCESS_KEY=你的火山引擎密钥
ANTHROPIC_API_KEY=你的Claude API密钥
```

### 4. 启动应用
```bash
npm run dev
```

应用将在 http://localhost:3000 启动

## 使用流程

### 1. 输入剧本
- 在左侧输入框粘贴漫剧剧本
- 支持包含角色对话、动作描述的格式
- 实时检查剧本格式和字数

### 2. 配置参数
- 关键帧数量：3-12张
- 画风风格：多种风格可选
- 漫剧类型：根据内容选择合适类型
- 分辨率和比例：根据需求选择

### 3. AI分析
- 点击"AI智能分析剧本"
- Claude自动解析剧本结构
- 识别角色、场景、关键帧位置

### 4. 生成分镜
- 先生成第一张图确认风格
- 满意后生成所有分镜图
- 可单独重新生成任意图片

### 5. 下载结果
- 单张下载或批量下载
- PNG格式，包含完整分镜信息

## 故障排除

### 常见问题

**Q: 图片生成失败**
- 检查火山引擎API密钥是否正确
- 确认账户余额充足
- 检查网络连接

**Q: 剧本解析失败**
- 检查Claude API密钥
- 确认剧本格式包含角色对话
- 剧本内容不要过短或过长

**Q: 人物一致性差**
- 在剧本中提供详细的角色外貌描述
- 确保角色名称在剧本中保持一致
- 可以重新生成第一张图调整基准

**Q: 分镜图不符合预期**
- 调整画风和类型选择
- 在剧本中增加更详细的场景描述
- 使用重新生成功能多次尝试

### 性能优化

- 建议关键帧数量不超过8张（减少等待时间）
- 使用2K分辨率平衡质量和速度
- 批量生成时请耐心等待，避免重复点击

## 技术支持

遇到问题请检查：
1. 控制台错误日志
2. API密钥配置
3. 网络连接状态
4. 剧本格式规范性
